# daemonizable-macros

Proc-macro companion crate for
[`daemonizable`](https://crates.io/crates/daemonizable). Don't depend on this
crate directly — `daemonizable` re-exports the `#[daemonizable::main]`
attribute behind its default-on `macros` feature.
